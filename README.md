# cv

## Requirements

* `make`
* `pdflatex`

## Usage

Run `make help` for help.  
Basically, you can either run `make` or `make ${language}` to make `.pdf` file.

## Supported languages

| Language      | tex           | pdf        |
|:--------------|:--------------|:-----------|
| eng (default) | cv.tex        | cv.pdf     |
| rus           | cv_rus.tex    | cv_rus.pdf |
